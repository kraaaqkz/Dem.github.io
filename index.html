Учебный пример: Генерация RSA-ключей

Ниже показан пошаговый процесс создания открытого и закрытого ключей по алгоритму RSA на маленьких числах.

Шаг 1: Выбираем два простых числа p и q

Простые числа — это те, что делятся только на 1 и самих себя.

Возьмём: p = 3, q = 5

Шаг 2: Вычисляем n = p × q

n = 3 × 5 = 15

Шаг 3: Вычисляем функцию Эйлера φ(n) = (p−1)(q−1)

φ(n) = (3−1) × (5−1) = 2 × 4 = 8

Шаг 4: Выбираем число e

Условия:

  • 1 < e < φ(n)
  • НОД(e, φ(n)) = 1 (взаимно простые)

Попробуем e = 3:

НОД(3, 8) = 1 → подходит!

Можно было взять e = 5 или e = 7 — тоже подошли бы.

Шаг 5: Находим закрытый ключ d

Ищем такое d, чтобы: (e × d) mod φ(n) = 1

Подставляем: (3 × d) mod 8 = 1

  • d = 1 → 3 × 1 = 3 → 3 mod 8 = 3
  • d = 2 → 6 mod 8 = 6
  • d = 3 → 9 mod 8 = 1

Найдено: d = 3

Итоговые ключи

Открытый ключ: (n, e) = (15, 3) → им можно зашифровать

Закрытый ключ: (n, d) = (15, 3) → им можно расшифровать

Проверка: шифрование и расшифровка

Пусть сообщение m = 2

Шифрование: c = me mod n = 23 mod 15 = 8

Расшифровка: m = cd mod n = 83 mod 15 = 512 mod 15 = 2

Сообщение восстановлено верно!

RSA: Шифрование и расшифровка сообщения

Учебный пример с маленькими числами для понимания принципа работы RSA.

Исходные данные

  • Простые числа: p = 3, q = 5
  • Модуль: n = p × q = 15
  • Функция Эйлера: φ(n) = (p−1)(q−1) = 8
  • Открытая экспонента: e = 3
  • Закрытая экспонента: d = 3

Шифрование сообщения

Пусть наше сообщение — число m = 2 (например, «2» означает «анализ готов»).

Условие: m < n → 2 < 15 →

Формула шифрования:

C = me mod n = 23 mod 15

Вычислим:

23 = 8 → 8 mod 15 = 8

→ Шифротекст: C = 8

Расшифровка сообщения

Используем закрытый ключ (n, d) = (15, 3).

Формула расшифровки:

M = Cd mod n = 83 mod 15

Вычислим:

83 = 512 → 512 mod 15 = 2

→ Расшифрованное сообщение: M = 2

Исходное сообщение восстановлено!

В реальных системах:
Число n состоит из 600+ десятичных цифр (около 2048 бит), что делает подбор ключей вычислительно невозможным даже для самых мощных компьютеров.
Этот пример использует tiny-числа только для обучения!
