🔍 Как «думает» наша нейросеть? Простыми словами

🤖 Нейросеть — это как команда из 2 экспертов (скрытый слой), которые смотрят на данные студента, каждый делает своё заключение, а потом главный эксперт (выходной слой) объединяет их мнения и выдаёт прогноз.

Сначала эксперты ошибаются, но после 1000 тренировок они учатся обращать внимание на то, что действительно важно:
📚 оценки за чтение и письмо, ✍️ подготовку к тесту и другие признаки.

CREATE TABLE Doctor (
    doctor_id INT PRIMARY KEY,

    age INT

-- Таблица МЕДСЕСТРА
CREATE TABLE Nurse (
    nurse_id INT PRIMARY KEY,
    name VARCHAR(100) NOT NULL,
    nurse_type VARCHAR(50),
    experience_years INT,
    age INT
);

-- Таблица ОПЕРАЦИЯ
CREATE TABLE Operation (
    operation_id INT PRIMARY KEY,
    operation_number VARCHAR(10),
    operation_date DATE,
    operation_type VARCHAR(100),
    start_time TIME,
    end_time TIME,
    doctor_id INT,
    nurse_id INT,
    FOREIGN KEY (doctor_id) REFERENCES Doctor(doctor_id),
    FOREIGN KEY (nurse_id) REFERENCES Nurse(nurse_id)
);
